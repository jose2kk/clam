# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue.
2. Use [GitHub Security Advisories](https://github.com/jose2kk/clam/security/advisories/new) to report the vulnerability privately.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within a few weeks. Once the issue is confirmed, a fix will be prepared and released as soon as possible.
